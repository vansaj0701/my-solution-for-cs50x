5 common Python libraries

Library Description
NumPy Used for numerical computing, handling large multi-dimensional arrays, and performing mathematical operations like linear algebra and statistics.
Pandas Provides powerful data manipulation and analysis tools, particularly for working with structured data like data frames (tabular data).
Matplotlib A plotting library used to create static, interactive, and animated visualizations like line graphs, bar charts, and histograms.
Scikit-learn A machine learning library offering tools for classification, regression, clustering, and more. Built on top of NumPy, SciPy, and Matplotlib.
Requests Simplifies sending HTTP requests and handling responses for tasks like web scraping or API communication.
