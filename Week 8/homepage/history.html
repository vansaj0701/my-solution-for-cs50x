Brief History of Python

Year Event
1980s Guido van Rossum began working on a project inspired by the ABC language.
1991 Python 0.9.0 was released with features like functions, exception handling, and modules.
1994 Python 1.0 was officially released, including features like lambda, map, and filter.
2000 Python 2.0 introduced list comprehensions and garbage collection via reference counting.
2008 Python 3.0 was released, emphasizing readability and consistency but breaking backward compatibility.
2010 Python 2.7 was released as the last major release of the Python 2 series.
2018 Python 3 became the preferred version as Python 2 was announced to reach its end-of-life.
2020 Python 2 officially reached its end-of-life on January 1, 2020.
2021 Python 3.10 was released, featuring pattern matching and other significant updates.
