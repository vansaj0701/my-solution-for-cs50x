Basic Syntax of Python

The syntax of Python is simple and designed to be readable. Below are key components of Python syntax:

1. Variables and Assignments

x = 10
name = "Alice"

2. Indentation

if x > 5:
 print("x is greater than 5") # Indented block

3. Data Types

# Integers, floats, and strings
a = 5     # int
b = 3.14   # float
c = "Hello"   # string

4. Control Statements

If-Else

if x > 5:  print("x is large") else:  print("x is small")

Loops

# For loop
for i in range(5):
 print(i)

# While loop
count = 0
while count < 5:
 print(count)
 count += 1

5. Functions

def greet(name):
 print(f"Hello, {name}!")

greet("Alice")

6. Input/Output

# Input
name = input("Enter your name: ")

# Output
print(f"Hello, {name}!")

7. Comments

# This is a single-line comment

"""
This is a
multi-line comment
"""
